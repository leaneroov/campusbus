# 시내버스 시간표 뷰어

## 주요 기능

- **버스 시간표 제공**: 필요에 따라 현재시간 기준 시간표 보기와 전체 보기를 제공
- **실시간 필터링**: 현재 시간 이후의 운행 예정 버스만 필터링하여 표시
